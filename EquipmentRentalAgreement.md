# Equipment Rental Agreement

## Purpose

Formal contract for the temporary rental of Table d’Adrian equipment to a client or venue.

## Fields

| Field                     | Description                                  |
| ------------------------- | -------------------------------------------- |
| **Agreement Date**        | \[Date]                                      |
| **Renter Name**           | \[Client / Venue Name]                       |
| **Event Date & Location** | \[Date and Venue Address]                    |
| **Rental Period**         | \[Start Date/Time] to \[End Date/Time]       |
| **Total Rental Fee**      | \[€ Amount]                                  |
| **Security Deposit**      | \[€ Amount] (e.g., 20% of replacement value) |

## Terms & Conditions (Key Clauses)

1. **Equipment List**: A detailed list of all rented equipment (e.g., Combi Oven, Sous-Vide Circulator, Induction Burners) is attached as Schedule A.

2. **Condition**: The Renter acknowledges receiving the equipment in good working order.

3. **Responsibility**: The Renter assumes full responsibility for the equipment during the Rental Period and agrees to return it in the same condition, reasonable wear and tear excepted.

4. **Security Deposit**: The Security Deposit will be held and used to cover any damage, loss, or excessive cleaning required. It will be returned within 7 days of the Rental Period end, provided the equipment is returned undamaged.

5. **Liability**: Table d’Adrian is not liable for any injury or loss arising from the Renter's use of the equipment.
