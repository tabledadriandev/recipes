# Chocolate cremeux

Notes: Anglaise base emulsified over chocolate; set as spoonable cream.
Salt %: 0.1
Shelf Life (days): 3
Storage: 0–3 °C
Temp °C: 83 cook; set 2–4 °C
Time: 4–6 h set
Type: Gel

### Method

1) Cook anglaise base to 82–83 °C.

2) Emulsify over chopped chocolate with immersion blender.

3) Portion; chill 4–6 h to set.

### Notes

- Blend from center to create a glossy, stable emulsion.
