# Panna cotta base

Notes: Gelatin 0.6–1.0%; balance with dairy fat.
Salt %: 0
Shelf Life (days): 3
Storage: 0–3 °C
Temp °C: Bloom 5–10; set 2–4 °C
Time: 4–6 h set
Type: Gel

### Method

1. Bloom gelatin; warm cream-sugar to dissolve.
2. Combine; flavor; strain; portion.
3. Chill 4–6 h to set.

### Notes

- Adjust gelatin for desired jiggle.
