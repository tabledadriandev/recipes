# Apple gel (agar)

Notes: Agar‑set clarified apple juice; blend to piping gel.
Salt %: 0.05
Shelf Life (days): 5
Storage: 0–3 °C
Temp °C: Boil 95–100; set 2–4 °C
Time: 2–4 h set
Type: Gel

### Method

1) Season clarified apple; whisk agar 0.6–0.8%.

2) Boil 1–2 min; set shallow.

3) Blend smooth; pass fine.

### Notes

- Add lemon for freshness.
