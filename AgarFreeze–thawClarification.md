# Agar Freeze–Thaw Clarification

## Technique

Simmer liquid with 0.4–0.6% agar for 1 min → set → freeze solid → hang in cheesecloth for 24–48 h.

## Parameters

- **Agar %**: 0.4–0.6%

- **Simmer Time**: 1 min

- **Hang Time**: 24–48 h

## Sensory Cues

- **Result**: Clean flavor, plant-based consommé.
