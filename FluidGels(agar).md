# Fluid Gels (Agar)

## Technique

Hydrate 0.8–1.2% agar → set → shear hard → pass 0.5 mm sieve → finish with 0.1% xanthan if needed.

## Parameters

- **Agar %**: 0.8–1.2%

- **Xanthan %**: 0.1% (optional finish)

- **Sieve**: 0.5 mm

## Sensory Cues

- **Result**: Smooth, fluid gel texture.
