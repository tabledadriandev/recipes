# Alcohol maceration (aroma extraction)

Applications: Tinctures, bitters, sprays
Confidence: 75
Corrections: Shorten time, dilute alcohol
Failure modes: Harshness, bitterness
Medium: Alcohol
Notes: Use neutral spirit 40–60%.
Sensory cue: Clean top notes, color extraction
Source: In-house
Standardized: No
Temperature range (°C): 2–20
Time range: 1–7 d
Tools used: Jar, filter
Type: Infuse

**Principle**

Ethanol extracts aromatic compounds rapidly at low temperatures.

- Parameters
    - Alcohol: neutral 40–60%
    - Temp: 2–20 °C
    - Time: 1–7 days (or SV 45–60 °C for 30–120 min)
- Goal
    - Aroma: clean top notes, target color
    - Flavor: minimal bitterness

<aside>
⚠️

Sensory benchmark: bright aroma without harsh ethanol bite

</aside>

- Test
    - Variable: proof level and time
    - Observe: extraction strength, bitterness
    - Acceptance: target intensity without harshness
- Uses
    - Veg: herb tinctures for dressings
    - Meat: spray for smoke-lift
    - Fish: citrus bitters finish
    - Dessert: infused syrups and gels
