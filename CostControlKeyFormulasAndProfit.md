# Cost Control: Key Formulas and Profit

## Key Formulas

1. **Portion Cost** = total recipe cost ÷ number of portions.

2. **Food-Cost %** = portion cost ÷ menu price × 100.

3. **Menu Price** = portion cost ÷ target food-cost %.

## Example Calculation

- Recipe cost = €4.00; target 30 %

- Menu price = 4 ÷ 0.30 = €13.33.

## Gross & Net Profit

- **Gross %** = 100 − food-cost %.

- **Net Profit** = gross % − labor/overhead.

- **Goal**: Keep both above target each week or adjust menu.
