# Agar clarification (freeze–thaw gel)

Applications: Fruit and veg consommés
Confidence: 75
Corrections: Increase agar 0.2–0.5%, longer set
Failure modes: Weak gel, haze
Medium: Water
Notes: Hydrate agar fully before set.
Sensory cue: Water-clear drip, clean aroma
Source: In-house
Standardized: No
Temperature range (°C): 90–95 hydrate; -18 freeze; 2–4 thaw
Time range: 12–24 h
Tools used: Agar, pot, freezer, sieve
Type: Clarify

### Method

1) Hydrate agar in juice at 90–95 °C; set to gel. 2) Freeze solid; thaw at 2–4 °C over filter to collect clear drip.

### Notes

- Sensory: water‑clear drip, clean aroma. Increase agar 0.2–0.5% if weak gel.
- Test: agar % and set time; acceptance = clear filtrate without haze.
- Uses: fruit and veg consommés.
