# Cucumber gel (agar)

Notes: Agar‑set cucumber juice; blend to gel purée for dots.
Salt %: 1
Shelf Life (days): 3
Storage: 0–3 °C
Temp °C: Boil 95–100; set 2–4 °C
Time: 2–4 h set
Type: Gel

### Method

1) Season cucumber juice; whisk in agar (0.6–0.8%).

2) Boil 1–2 min; pour to set.

3) Blend set gel to smooth; pass fine.

### Notes

- Add lemon for brightness.
